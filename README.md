# Brand-wise Product Review Sentiment Analysis

## 📌 Project Overview
This project performs sentiment analysis on customer reviews of wireless earbuds and classifies them into:

- Positive
- Neutral
- Negative

The project also provides brand-wise sentiment analysis and visualization for different brands such as boAt, Noise, OnePlus, JBL, and Realme.

---

## 🚀 Features
- Sentiment classification of customer reviews
- TF-IDF based text vectorization
- Machine Learning model training using Logistic Regression
- Evaluation using Accuracy, Classification Report, and Confusion Matrix
- Custom customer review prediction
- Brand-wise sentiment analysis
- Data visualization using charts and graphs

---

## 🛠️ Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn
- Google Colab / Jupyter Notebook

---

## 📂 Dataset
The dataset contains:
- Brand name
- Product name
- Review text
- Rating
- Sentiment label

A synthetic but realistic wireless earbuds review dataset was used for this project.

---

## ⚙️ Machine Learning Workflow
1. Import Libraries
2. Load Dataset
3. Convert Text into TF-IDF
4. Train ML Model
5. Evaluate Model
6. Predict Customer Review
7. Brand-wise Sentiment Analysis
8. Visualization

---

## 📊 Output
The model predicts whether a review is:
- Positive
- Neutral
- Negative

It also compares customer sentiment across multiple brands.
